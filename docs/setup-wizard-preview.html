Setup Wizard — Visual Preview

First-time shop owner experience. Toggle between deployment modes.

Self-hosted: Shop runs Bizarre CRM on its own LAN/box. Owner installs via setup.bat or Docker. Logs in with default admin/admin123. No tenant slug, no signup, no trial. Free forever.
