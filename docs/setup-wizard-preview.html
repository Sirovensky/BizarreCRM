Setup Wizard — Visual Preview

First-time shop owner experience. After this wizard, shop is fully operational.

Deployment mode: Estimated total: ~12-15 min (skip non-essentials to finish in ~5)

Self-hosted: Shop runs Bizarre CRM on its own LAN/box. Owner installs via setup.bat or Docker. Logs in with default admin/admin123. No tenant slug, no signup, no trial. Free forever.

Annotations

  • Schema: writes repair_prices(device_model_id, repair_service_id, labor_price) from migration 010 (composite key — already per-device).
  • Tier expansion: server back-fills 2,436 rows: 12 services × 203 device models. Each device's tier looked up by device_models.released_year.
  • Tier rules (configurable in Settings): A = released ≤ 2yr ago, B = 3-5yr, C = >5yr. Re-evaluated nightly.
  • Per-device override: "Per-device matrix" tab opens grid view (203 rows × 12 cols) for hot models needing custom prices.
  • Auto-margin: when on, labor = (target_profit + supplier_part_cost). Catalog scrape (Mobilesentrix/PhoneLcdParts via cheerio per CLAUDE.md) refreshes part costs nightly.
  • $0 sentinel: disables that service for that tier (CROSS31 manual-price still avail).
  • Skip: empty catalog, manual price prompt per ticket.
