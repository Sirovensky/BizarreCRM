# Inventory

## Adding items

Tap **+** on the Inventory screen. Fill in the name, SKU, cost, sell price,
quantity, and reorder level. Save to add the item to your catalog.

## Barcodes

Each item can have a barcode (Code 128 or QR). Tap the barcode icon on the
item detail to view or print a label. Use the camera icon on the Inventory
list to scan an item quickly.

## Stock adjustments

Open an item and tap **Adjust Stock** to add or remove units (receiving,
damage write-off, manual correction). All adjustments are logged for audit.

## Reorder alerts

Set a **Reorder Level** on each item. When quantity falls below the level,
the item appears in the low-stock report and triggers a notification if
configured in **Settings → Notifications**.

## Purchase orders

Create a purchase order in **Inventory → Purchase Orders**. Add line items,
set expected delivery date, and send to the supplier. When stock arrives,
tap **Receive Items** to update quantities automatically.

## Stocktake

Go to **Inventory → Stocktake** for a physical count workflow. Count each
item offline, then commit to reconcile against system quantities.
