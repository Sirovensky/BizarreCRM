Design notes — iPad · Liquid Glass

Decisions captured · 2026-05-04

Dynamic dashboard

  • 5-col tile grid · drag, drop, resize 1×1 → 3×3 (tablet ceiling)
  • Edit mode: long-press tile or ⚙ Customize capsule
  • SwiftUI: LazyVGrid + .draggable() / .dropDestination()
  • Cross-device sync lives in Settings → Dashboard → Sync (not on toolbar)
  • Layout placement diverges per platform; widget catalog shared

Sidebar nav (9 items)

  • Floating glass platter · cream selection halo
  • + Comms (after Tickets) · unified customer inbox
  • Comms badge = unread count from SMS / calls / voicemail / email / chat
  • Bell removed — system events surface in Needs-attention card

Removed / Added

  • Removed: floating "2 new messages" card · "store opens in 19 min" subtitle · Top repairs widget
  • Added: Avg ticket KPI ($137) · Comms sidebar item · Customize capsule (sync moved to Settings)

Liquid Glass conventions

  • Chrome (sidebar, capsules) = real translucent glass + blur
  • Content tiles = SOLID surface-elev for readability
  • No saturated radial blobs — barely-visible ambient + grain + vignette
  • Modeled on Apple Wallet / Photos / Maps + Square / Toast iPad POS
