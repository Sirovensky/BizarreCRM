Design principles · research-agent consensus

Search in nav-bar drawer

.searchable(placement: .navigationBarDrawer(displayMode: .always)). Pinned under title, always visible — zero tap to begin. iOS 26 auto-moves this to bottom; `.always` keeps it at top on current iOS.

Tender in safe-area

.safeAreaInset(edge: .bottom). 56pt min height, within 120pt of home indicator per thumb-reach research. Cream gradient, on-primary #2b1400 text = AAA 15.23:1.

Sheet detents, not modals

Line edit = .height(320). Customer picker = .medium. Receipt = full-screen. Drag indicator on every multi-detent sheet.

Barlow Condensed hero numerics

$274.51 at 32–42pt with font-variant-numeric: tabular-nums. Size-locked — Dynamic Type caps at .accessibility2 for layout-sensitive text.

Two primary actions on home

New sale + New repair ticket. Separate upfront paths — not a convert-at-tender flow. Matches RepairDesk/RepairShopr ergonomics.

Customer attach optional

Square/Shopify pattern: attach any time before tender, never blocking. Single-prompt at Charge tap if missing; 2s dismissible.

Quick-amount strip

Above numpad on every cash screen. Exact · +$5 · +$10 · +$20. Removes the #1 cash entry error (wrong-denom field).

Dynamic Island live activity

ActivityKit.Activity<POSCartAttributes>. Compact leading = cart icon + count, trailing = total. iPhone 14 Pro+. Cleared immediately on Paid.

Liquid Glass on chrome only

Topbar, cart strip, sheet, tender safe-area, connectivity chip. Content (cart rows, catalog tiles, receipt list) on solid surface-solid. Max 4–5 non-contained glass panels — GPU budget.

Portrait lock

Matches Square, Shopify POS, Toast Go. Numpad + tender button + cart are portrait-optimized. Use UISupportedInterfaceOrientations.

Haptics

.sensoryFeedback iOS 17+. Added to cart = .impact(.light). Scan success = .success. Tender complete = .success + .impact(.heavy, 0.8).

App Intents / Siri

NewSaleIntent, AddItemIntent(sku:), FindCustomerIntent(query:). Auto-appear in Shortcuts, Spotlight, Siri.

Self-contained preview · no JS · no network. Tokens mirror ios/Packages/DesignSystem/Sources/DesignSystem/BrandColors.swift + BrandFonts.swift + BrandSpacing.swift. Ports directly to SwiftUI.

Light mode · same flow

Exact same 11 frames rendered under the light-theme token set. Cream primary #fdeed0 stays constant. Surface tokens flip to near-white; semantic colours (success / warning / error / teal) are darkened for WCAG AA contrast on white-ish backgrounds.
