BIZARRE CRM · DASHBOARD STUDY

5 surfaces, 2 modes, 1 feel

Dashboard rendered for web, iPhone, iPad, Android phone, Android tablet — each in dark + light. Same content (greeting, KPIs, alerts, activity) so functional parity is visible. Each respects its platform: iOS = Liquid Glass (translucency, layered blur), Android = Material 3 Expressive (asymmetric tiles, big radii, FAB), Web = web. Mockups, not specs — real dashboards have more.

Cream / Caramel primary Teal secondary Burgundy tertiary Warm-graphite ramp

Web

1320 × 800 · React + Tailwind

Standard browser layout. Persistent left sidebar (always visible), top header with global search, dense content area. Designed for keyboard + mouse: hover states, multi-column dense data, expansive widgets. Light mode favored when used in office; dark mode for after-hours.

Android phone · Material 3 Expressive

412 × 915 · Compose Material3

Material 3 Expressive: 24px+ corner radii, asymmetric tile sizes (Sales is 2× wide), bold typography, primary-tone hero block (no glass — solid brand fills). Bottom navigation bar with 5 destinations. Extended FAB — default 📷 Snap to ticket (camera → attach photo to in-progress ticket). User-configurable in Settings → Dashboard quick action; alt options include 🎤 Voice note · 🔍 Scan IMEI · 📞 Call top customer · 📤 Send quote · 📦 Receive part · ⏱️ Start labor timer · 🏁 Advance ticket stage · 💰 Open drawer. POS still owns ticket creation; FAB is for capture-during-walkaround. More color saturation than iOS — primary cream/caramel fills are intentional.

Platform tells

Web

  • • Browser chrome + URL bar
  • • Persistent left sidebar (always visible)
  • • Dense KPI grid · 4-up
  • • Hover states · ⌘K search
  • • Top header with notification badges
  • • Scrolling content area

iOS · Liquid Glass

  • • Translucent toolbars (backdrop-filter blur)
  • • Tinted gradient backgrounds bleed brand color
  • • Frosted glass cards · thin 0.5px borders
  • • Floating tab bar / NavSplitView
  • • Dynamic Island · home indicator
  • • Cream wash through translucent layers

Android · M3 Expressive

  • • Big radii (24-28px) · solid container fills
  • • Asymmetric grid (Sales = 2× wide hero)
  • • Floating Action Button (extended on tablet)
  • • Bottom nav (phone) · Navigation Rail (tablet)
  • • Pill-shaped active indicators on nav
  • • Heavier brand color presence — primary fills
