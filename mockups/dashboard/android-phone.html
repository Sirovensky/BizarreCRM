Android phone · Material 3 Expressive

412 × 915 · Compose Material3

Material 3 Expressive: 24px+ corner radii, asymmetric tile sizes (Sales is 2× wide), bold typography, primary-tone hero block (no glass — solid brand fills). Bottom navigation bar with 5 destinations. Extended FAB — default 📷 Snap to ticket (camera → attach photo to in-progress ticket). User-configurable in Settings → Dashboard quick action; alt options include 🎤 Voice note · 🔍 Scan IMEI · 📞 Call top customer · 📤 Send quote · 📦 Receive part · ⏱️ Start labor timer · 🏁 Advance ticket stage · 💰 Open drawer. POS still owns ticket creation; FAB is for capture-during-walkaround. More color saturation than iOS — primary cream/caramel fills are intentional.

Design notes — Android phone · M3 Expressive

Decisions captured · 2026-05-04

Snap to ticket — Extended FAB

  • M3 Extended FAB · cream/caramel fill · 18px radius
  • Default action: 📷 Snap to ticket (configurable in Settings)
  • Alt actions: 🎤 Voice note · 🔍 Scan IMEI · 📞 Call · 📤 Quote · 📦 Receive part · ⏱ Labor timer · 🏁 Advance stage · 💰 Drawer

Bottom nav — 5 destinations

  • Home · Tickets · POS · Comms · More
  • Comms = unified customer inbox · pill highlight on active
  • Customers / Inventory / Reports / Settings live in More
  • M3 spec: 3-5 destinations max

Dynamic dashboard

  • 2-col tile grid · phone constraint
  • Phones: reorder + toggle only (no resize)
  • Compose: Calvin-LL/Reorderable + LazyVerticalGrid
  • ⚙ icon button in header opens edit mode
  • Cross-device sync lives in Settings → Dashboard → Sync (not on chrome)

Removed / Added

  • Removed: Top repairs widget · Customers wide-tile · "Top repairs" mini-tile
  • Added: Avg ticket KPI ($137) · Comms tab · ⚙ Customize button · Customer trends report
  • Bell removed — system events surface in Needs-attention card
