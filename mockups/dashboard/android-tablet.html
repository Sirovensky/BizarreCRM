Design notes — Android tablet · M3 Expressive

Decisions captured · 2026-05-04

Dynamic dashboard

  • 4-col asymmetric tile grid · drag, drop, resize 1×1 → 3×3 (tablet ceiling)
  • Hero tile (Sales) defaults to 2×2; user can resize
  • Compose: Calvin-LL/Reorderable + LazyVerticalGrid
  • ⚙ Customize pill in toolbar opens edit mode
  • Cross-device sync lives in Settings → Dashboard → Sync (not on toolbar)

Navigation Rail (M3)

  • 7 destinations · vertical icon stack · pill highlight on active
  • + Comms after Tickets · burgundy badge for unread
  • Order: Home · POS · Tickets · Comms · Customers · Inventory · Reports · Settings

Removed / Added

  • Removed: Peak hour from Sales hero · Top repairs widget
  • Added: Comms rail item · ⚙ Customize pill in toolbar (sync moved to Settings)
  • Avg ticket KPI ($137) already present

M3 Expressive conventions

  • Big radii (24-28px) · solid container fills
  • Asymmetric grid · primary-tone hero (cream/caramel)
  • Bold brand color presence — primary fills, not glass
  • Reads premium-tablet, not phone-stretched
