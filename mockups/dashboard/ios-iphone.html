Design notes — iPhone · Liquid Glass

Decisions captured · 2026-05-04

Snap to ticket — bottom accessory

  • Liquid Glass capsule above tab bar · cream tint + halo
  • Pattern: SwiftUI tabViewBottomAccessory (iOS 26)
  • Apple precedent: Music · Podcasts · Health · Maps
  • Tap → camera capture → attach photo to in-progress ticket

Tab bar — 5 destinations

  • Dashboard · Tickets · POS · Comms · More
  • Comms = unified customer inbox (SMS, calls, voicemail, email, chat) · badge = unread
  • Customers / Inventory / Reports / Settings live in More
  • HIG max 5 tabs respected

Dynamic dashboard

  • 2-col tile grid · phone constraint
  • Phones: reorder + toggle only (no resize)
  • Edit mode entered via long-press tile or Settings
  • Cross-device sync lives in Settings → Dashboard → Sync (not on chrome)
  • Layout placement diverges per platform; widget catalog shared

Bell vs Comms · Removed widgets

  • Bell removed from iPhone toolbar — system events surface in Needs-attention card
  • Comms = customer messages (tab bar)
  • Removed: Top repairs · Peak hour
  • Added: Avg ticket KPI ($137)
