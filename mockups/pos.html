POS — Point of Sale

Point-of-sale rendered for web, iPhone, iPad, Android phone, Android tablet — each in dark + light. 9 screens per surface showing the complete sale flow: Entry → Categories → Manufacturers → Devices → Parts & Repairs → Search Results → Custom Part → Tender → Receipt. Each respects its platform: iOS = Liquid Glass, Android = M3 Expressive, Web = web.

Web

1320 × 800 · React + Tailwind

Persistent left sidebar, top header bar. POS lives as a page within the CRM — same shell as Dashboard. Content area splits: catalog browsing left, cart panel right (always visible). Drill-down flow: categories → manufacturers → devices → parts/repairs. Keyboard-first: barcode scanner, F-key shortcuts, tab navigation.
