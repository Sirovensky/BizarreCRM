Design language cheatsheet

  • Liquid Glass (iOS). Five cues: real refraction via SVG feDisplacementMap, specular rim gradient, saturation pickup saturate(180%), low bg alpha 0.08, inner top highlight. Concentric corners 28 → 22 → 14. Warm multi-radial orb background so glass has color to bend.
  • M3 Expressive α (Android). LargeFlexibleTopAppBar, LinearWavyProgressIndicator, shape-morph buttons (28 → 16 on press), MaterialShapes 9-sided cookie, FAB Menu, Split Button, emphasized type. No glassmorphism — explicit ban in Theme.kt.
  • Shared brand. Cream #FDEED0 primary stays mobile-distinct. Surface ramp now Zinc neutral aligned with bizarrecrm.com web — #09090B → #18181B → #27272A dark, #FAFAFA → #F4F4F5 light. Brand orange #F58220 available as accent. Barlow Condensed display, Inter body, JetBrains Mono for OTP / tokens.
  • Tablet parity. iPad uses NavigationSplitView. Android tablet uses nav rail + supporting pane. Neither is an upscaled phone.
  • Gate. HTML mockups only. Tag both apps before any Compose / SwiftUI implementation.
